# Trae Configurations

This repository contains Trae IDE profile configurations for different development environments.

## Profiles

### Node.js.code-profile
A Trae IDE profile optimized for Node.js development, including recommended extensions, settings, and workspace configurations.

### Python.code-profile
A Trae IDE profile optimized for Python development, including recommended extensions, settings, and workspace configurations.

## Usage

1. Open Trae IDE
2. Go to the profiles section in settings
3. Select the option to import a profile
4. Choose the desired profile file from this repository
5. Follow the prompts to complete the import

## Features

- Pre-configured settings for optimal development experience
- Recommended extensions for each language ecosystem
- Workspace configurations tailored to common project structures
- Debugging and testing configurations
